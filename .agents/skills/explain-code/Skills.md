# Canvas API キャラクター描画スキル集

Canvas APIでキャラクター（特に妖怪・精霊系）を描画する際の実装ガイドライン。

---

## Skill: 有機的な図形描画 (Organic Shape Drawing)
- **概要**: キャラクターを描画する際、`arc` (真円) や `fillRect` (四角形) の単純な組み合わせを固く禁じる。`bezierCurveTo` や `quadraticCurveTo` などのパス描画を駆使し、滑らかで有機的な輪郭を構築すること。
- **実装要件**:
  - 顔の輪郭、髪の毛、服装のシワなどは、必ずベジェ曲線を用いて自然な形状（アシンメトリーな柔らかい線など）で描画すること。
  - 目や口などの顔のパーツも、単純な図形ではなく、表情のニュアンスが出るようにパスで丁寧に描画すること。

## Skill: リッチな陰影と立体感 (Rich Gradient & Shading)
- **概要**: のっぺりとした単色塗りつぶし（Solid Color）を避け、グラデーションを用いてキャラクターにリッチな立体感と質感を付与する。
- **実装要件**:
  - `createRadialGradient` や `createLinearGradient` をベースカラーに適用し、光の当たり具合（ハイライト）と落ち影（ドロップシャドウ）を表現すること。
  - キャラクターのパーツごとに階層（レイヤー）を意識し、奥にあるパーツは暗めに、手前にあるパーツは明るめに描画するロジックを組むこと。

## Skill: 妖怪・精霊のオーラとエフェクト (Ethereal Yokai Aura)
- **概要**: 妖怪の神秘的・霊的な雰囲気を演出するため、Canvasのブレンドモードやパーティクルを活用した動的なエフェクトを実装する。
- **実装要件**:
  - `globalCompositeOperation = 'screen'` または `'lighter'` を使用し、キャラクターの背後や周囲に多重の半透明な発光（グロウエフェクト）を描画すること。
  - ただ静止させるのではなく、`Math.sin()` を用いたゆっくりとした上下の浮遊アニメーションや、周囲を漂う人魂（小さな光のパーティクル）をプログラムで生成すること。
