ゲーム開始待ち / Waiting to start

📋 ブロックストライゴン ルール

目的 / Objective

22種類の三角形ピースをできるだけ多く盤面に置き、残ったピースのセル数が最も少ないプレイヤーが勝利。

配置ルール / Placement Rules

  • 最初の1手は自分のコーナーセルを覆うように置く
  • 2手目以降は自分のピースの頂点（コーナー）に接するように置く
  • 自分のピース同士は辺では接触不可
  • 相手のピースとは辺・頂点どちらで接触してもOK

ピース操作 / Piece Controls

  • ↺ / ↻ ボタン — 60° 回転（三角グリッド専用）
  • ⇔ ボタン — 左右反転
  • パネルのミニ図をクリック → ピース選択、盤面クリックで配置
  • キーボード: R=回転, F=反転

パスと終了 / Pass & End

  • 有効な配置がない場合は「パス」を押す
  • 全プレイヤーが連続パス → ゲーム終了

得点計算 / Scoring

  • 残りピースのセル数合計が少ないほど高得点
  • 全ピースを使い切ると +20 点
  • 最後がM1（1セル）なら追加で +5 点

ゲーム終了 / Game Over
