# Unity版 将棋RPG（shogi_rpg.html 移植・最小コア）

このフォルダは、`shogi_rpg.html` の「盤面（9×9）＋将棋駒移動＋3マッチ消去＋重力落下＋プレイヤー/敵の交互ターン＋ステージ制（妖怪HP）」のコアを、Unity（C#）で動く形に移植したものです。

## できていること（この移植に含めたもの）
- 9×9 固定盤面
- 将棋駒ごとの移動（`shogi_rpg.html` の `PIECES[type].moves` を踏襲）
- 3マッチ検出（横・縦・斜め2方向、いずれも3つ一致）
- 消去 → 落下（下方向）→ 補充 → 連鎖（コンボ）
- プレイヤーターン→敵AIターン（`findBestMove` 相当で最良手を選ぶ）
- ステージ1〜10の妖怪（名前/HP/攻撃倍率）

## 省略したもの（必要なら次で追加できます）
- ショップ/ガチャ/インベントリ/アイテム効果
- 実績/デイリーミッション/ランキング/図鑑/ストーリー表示
- SVG演出、効果音（Web Audio相当）
- 成駒札/成り駒（※駒タイプ自体は追加しやすい構造にしてあります）

## 使い方
1. Unityで新規2Dプロジェクトを作成（Unity 2021 LTS 以降推奨）
2. このリポジトリの `UnityShogiRPG/Assets` を、あなたのUnityプロジェクト直下の `Assets` にコピー（または `Assets` 配下にドラッグ＆ドロップ）
3. Playすると、自動で `ShogiRPGGame` が起動します（シーン不要）

うまく起動しない場合:
- 空のGameObjectを1つ作り、`ShogiRPGGame`（`Assets/ShogiRPG/Scripts/ShogiRPGGame.cs`）をAdd Componentしてください。

## 操作
- 盤面をクリックして駒選択 → 移動可能マスをクリックで入れ替え
- 連鎖が止まると敵が自動で手を指します
