# Gradius - Boss Rush Mode

## 概要
gradius.htmlは、元々のgradius.jsx（React + Canvas API）から最も重要な機能である**ボス戦闘システム**とステージシステムを優先して実装したシングルページゲームです。

## 特徴
- ✅ **7つのボスステージ**: 火龍、モアイ、ツタンカーメン、エイリアン、アメーバ、コロナ、要塞
- ✅ **ボスラッシュモード**: 各ステージで1つのボスと対戦して撃破できれば次のステージへ進行
- ✅ **Web Audio API BGM**: 各ステージに固有のバックグラウンドミュージック
- ✅ **リアルタイムボスHP表示**: ボス画面上部にHP ゲージを表示
- ✅ **スコアシステム**: ボス撃破で1000ポイント獲得
- ✅ **3ライフシステム**: 敵弾に被弾するとライフ減少

## 操作方法

| キー | 機能 |
|------|------|
| **↑/↓/←/→** | プレイヤー移動 |
| **Z / スペース** | 射撃 / 連続射撃 |
| **End** | 次のステージへスキップ（デバッグ用） |

## ゲームの流れ

1. **STAGE 1 開始**: ツイン火龍が画面右側に出現
2. **ボス戦闘**: プレイヤーが赤い敵弾を避けながら、黄色の弾でボスを攻撃
3. **ボス撃破**: ボスのHP が 0 になるとステージクリア
4. **ステージ進行**: 自動で次のステージへ（1秒の待機時間）
5. **STAGE 7 クリア後**: "ALL STAGES CLEARED!" 表示で全ゲーム完了

## 技術構成

- **HTML5 Canvas API**: ゲーム描画
- **Web Audio API**: BGM・効果音生成
- **Pure JavaScript**: フレームワーク不使用
- **CSS Grid + Flexbox**: UI レイアウト

## ゲーム内UI

### ヘッドアップディスプレイ (HUD)
画面左上に以下の情報が表示されます：
- **STAGE**: 現在のステージ番号（1～7）
- **SCORE**: 累積スコア
- **LIVES**: 残りライフ数
- **BOSS HP**: ボスの現在HP

### ステージ情報
ステージ開始時に画面中央に「STAGE X: [ボス名]」と2秒間表示されます。

## デバッグ・便利機能

### ステージスキップ
Endキーを押すと現在のステージをスキップして次のステージへ進みます（テスト用）。

### コンソール操作例
ブラウザ開発者ツール (F12) でコンソールを開き、以下を実行可能：
```javascript
gameState.lives = 10;  // ライフを10に設定
gameState.score = 5000;  // スコアを5000に設定
gameState.stage = 5;  // STAGE 5に変更
```

## トラブルシューティング

### ゲームが開かない
- ブラウザのコンソール (F12 → Console) を開いて、エラーメッセージを確認してください。
- 最新のChrome, Firefox, Safari, Edge で動作が確認されています。

### BGM が鳴らない
- ブラウザの音声設定を確認してください
- Web Audio API はユーザー操作後に有効化される仕様のため、最初の操作後に音が出ます

### ボスが動かない
- フレームレートが低い可能性があります。他のアプリを閉じてみてください
