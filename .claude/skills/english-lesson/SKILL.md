# /english-lesson — 英語学習セッション

english-teacher エージェント（ネイティブ講師・日英バイリンガル指導）でのレッスンを定型化する。制作パイプラインとは独立した単発ユーティリティ。

## 手順

### 1. モードとレベルを確認
深澤の依頼から以下を特定する（不明ならAskUserQuestionで確認）:
- **モード**: ①英会話・スピーキング練習（ロールプレイ/自由会話） ②英作文・メール添削 ③文法・語彙の解説 ④発音・リスニング指導
- **CEFRレベル**: A1〜C2（過去のレッスン記録があればそのレベルを引き継ぐ。初回はB1想定で開始し会話の中で調整）
- **テーマ**（あれば）: 仕事メール・旅行・技術英語など

### 2. english-teacher エージェントを起動
Agent tool で `english-teacher` を起動し、モード・レベル・テーマを冒頭で明示して引き渡す。添削モードの場合は対象英文を全文渡す。

エージェントの指導原則（定義済み・変更しない）: 褒めてから直す / 間違いを歓迎する / Good points → Corrections → Native version の添削構成。

### 3. 学習記録をVaultへ（セッション終了時）
`obsidian-vault/01-Daily/YYYY-MM-DD.md` に1〜2行で記録する（Daily圧縮ルール準拠）:

```
- [english] モード②添削: ビジネスメール。学び: "Please kindly" は冗長 → "Please" のみ。推定レベルB1+
```

- 繰り返し間違えるパターン（3回目以降）は `04-Knowledge/english-weak-points.md` を作って集約する（次回レッスンの冒頭で講師に渡すと復習になる）

## 原則
- レッスン中の会話は記録しない（記録するのは「学び」と「レベル」のみ）
- 学習モチベーション維持を最優先（english-teacher の基本姿勢に合わせ、記録も成長が見える形で書く）
