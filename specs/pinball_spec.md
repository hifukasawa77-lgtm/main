# ピンボール / Pinball — 仕様書（ミニ仕様）

GameKit（`gamekit/gamekit.js`）上に実装する縦型シングルキャンバスのピンボール。

## 基本情報
- ファイル: `pinball.html`（リポジトリ直下、`gamekit/template.html` ベース）
- キャンバス: 540 × 760（縦長テーブル）
- 土台: GameKit（Engine / Scene / Input / Sfx / Particles / UI / Save / MathX / Collision）
- デザイン: 黒背景 `#05070d` + シアン `#22d3ee` / パープル `#a78bfa`、Glassmorphism、日英バイリンガルUI、サイバーパンク演出なし
- 画像アセット: なし（全てCanvasコード描画＝プロシージャル）

## ジャンル / コアループ
物理ベースのピンボール。重力で落下するボールを左右フリッパーで弾き返し、バンパー・スリングショット・ドロップターゲットに当ててスコアを稼ぐ。残ボールを使い切るまでの合計スコアを競う。

## 操作 / Controls
- 左フリッパー: ← / A / Z / 画面左半分タップ
- 右フリッパー: → / D / M / 画面右半分タップ
- 発射（プランジャー）: ボールがレーン待機中に Space または画面長押しで力をチャージ → 離して発射

## 勝敗条件
- ボール数: 3
- ボールがドレイン（最下部中央）から落ちると1つ消費
- 残ボール0で GAME OVER → スコアをハイスコアと比較・保存（`GameKit.Save`）

## 画面構成 / テーブル要素
1. アーチ状の上壁＋左右壁＋右側シューターレーン（プランジャー発射路）
2. ポップバンパー ×3（命中 +100、エネルギー付与＋フラッシュ）
3. スリングショット ×2（フリッパー上の角面、命中 +60、強めにキック）
4. ドロップターゲット バンク ×3（命中 +250 で消灯、3枚クリアで +1500 ボーナス＆リセット）
5. 左右フリッパー（回転する剛体、角速度をボールへ伝達）
6. 下部ファネル → 中央ドレイン
7. HUD: SCORE / BALLS / HIGH SCORE、チャージメーター、メッセージ

## 物理仕様
- 重力 + 速度積分、240Hz相当のサブステップでトンネリング防止
- 衝突: 円×線分（最closest point）で反射。壁は反発0.55、バンパー/スリングは反発>1で加速
- フリッパー: ピボット回転。接触点の角速度をボール速度へ加算（弾き返し）
- 最大速度クランプ

## 必要アセット一覧
- なし（プロシージャル描画のみ）。SEは `GameKit.Sfx`（beep/noise/jingle）で生成
