## 1. システム構成図

```
train-zukan.html          ← エントリポイント（単一HTMLファイル）
    │
    ├── <style>           ← インラインCSS（全画面分）
    │
    ├── data/trains.js    ← 車両マスターデータ（JS変数として定義）
    │       └── window.TRAINS_DATA = { vehicles: [...], gacha: {...} }
    │
    ├── <script>          ← インラインJS（全機能）
    │       ├── StorageManager    localStorage の読み書きラッパー
    │       ├── GameState         ゲーム状態オブジェクト（コイン・コレクション等）
    │       ├── ScreenManager     画面遷移・表示制御
    │       ├── HomeScreen        ホーム画面描画・デイリーボーナス
    │       ├── ZukanScreen       図鑑画面描画
    │       ├── GachaScreen       ガチャ演出Canvas制御
    │       ├── StampRallyScreen  スタンプラリー地図Canvas・ミニゲーム
    │       ├── QuizScreen        たんていクイズ描画・ロジック
    │       ├── SugorokuScreen    たびすごろく盤面Canvas・ロジック
    │       ├── TrainAnimator     車両走行アニメーション共通処理
    │       └── SoundManager      Web Audio API 効果音
    │
    └── assets/trains/    ← 車両グラフィック格納
            ├── shinkansen-nozomi.png   （256×128px PNG 透過）
            ├── shinkansen-hayabusa.png
            ├── ...（60両分）
            └── silhouette-mask.js      （シルエット生成用黒マスク）
```

## 5. localStorage 詳細設計

| キー | 型 | 初期値 | 内容 |
|---|---|---|---|
| `train-zukan-v1` | JSON配列 | `[]` | 収集済み車両IDの配列 `["shinkansen-nozomi", ...]` |
| `train-coins-v1` | 数値文字列 | `"100"` | 所持コイン数（初期100コイン） |
| `train-daily-v1` | 文字列 | `""` | 最終ログイン日 `"2026-06-03"` (YYYY-MM-DD) |
| `train-pity-v1` | 数値文字列 | `"0"` | ピティカウンター（0〜9、10でSR確定） |
| `train-stamps-v1` | JSON配列 | `[]` | 取得済みスタンプ駅IDの配列 `["tokyo", ...]` |
| `train-sugoroku-v1` | JSONオブジェクト | `{"pos":1,"laps":0}` | すごろく進行状態 `{pos: 現在マス番号, laps: 周回数}` |
| `train-quiz-v1` | JSONオブジェクト | `{"date":"","count":0,"correct":0}` | クイズ記録 `{date: 今日の日付, count: 本日出題数, correct: 正解数}` |
| `train-mute-v1` | 文字列 | `"false"` | ミュート状態 `"true"` / `"false"` |
| `train-titles-v1` | JSON配列 | `[]` | 取得済み称号IDの配列 |

### localStorage アクセス設計

```
StorageManager
├── get(key): JSON.parse(localStorage.getItem(key)) — try-catch付き
├── set(key, value): localStorage.setItem(key, JSON.stringify(value))
├── getCoins(): 数値を返す
├── addCoins(amount): コイン加算
├── spendCoins(amount): コイン減算（残高チェック付き、負にならない）
├── getCollection(): 配列を返す
├── addToCollection(vehicleId): 重複チェックして追加
├── isCollected(vehicleId): boolean
├── getDailyBonus(): {lastDate, todayDate} を返す
├── recordDailyBonus(): 今日の日付を記録
├── getPity(): 数値を返す
├── incrementPity(): ピティカウンターをインクリメント
└── resetPity(): ピティカウンターをリセット
```
