Google Analytics (GA4) 測定ID 取得手順書

測定ID（G-XXXXXXXXXX 形式）をGoogle Analyticsの管理画面から取得します。

1
Google Analytics にアクセスする

ブラウザで Google Analytics を開きます。

URL: analytics.google.com

サイトに使用しているGoogleアカウントでログインしてください。
すでにログイン済みの場合はホーム画面が表示されます。

2
左下の「管理」（歯車アイコン）をクリックする

ホーム画面の左サイドバー最下部に歯車アイコン「管理」があります。

メニュー
🏠 ホーム
📊 レポート
🔍 探索
📢 広告
⚙️ 管理　　← ここをクリック
場所: 画面の左下隅に固定されています。スクロールしなくても表示されています。
3
「データストリーム」をクリックする

管理画面が開くと、左側に「プロパティ」列が表示されます。
その中の データストリーム をクリックします。

アカウント
▶ hide の部屋
プロパティ
プロパティ設定
データ収集と修正
データストリーム　← ここ
イベント
コンバージョン
4
ウェブストリームをクリックする

データストリームの一覧が表示されます。
「ウェブ」と表示されているストリーム（サイトURLが書かれたもの）をクリックします。

データストリーム
🌐 ウェブ　← これをクリック
hifukasawa77-lgtm.github.io/main/
ストリーム名: hide の部屋
複数のストリームがある場合: 「hifukasawa77-lgtm.github.io」と表示されているものを選んでください。
5
「測定ID」を確認・コピーする

ストリームの詳細画面が開きます。右上に「測定ID」が表示されています。

ウェブストリームの詳細
ストリーム名
hide の部屋
ストリームURL
https://hifukasawa77-lgtm.github.io/main/
測定ID　← ここ
G-XXXXXXXXXX
↑ この値をコピーする
コピーする値（形式）
G-XXXXXXXXXX
「G-」から始まる10〜12文字の英数字です
コピー方法: 測定IDの右側にある コピーアイコン（📋） をクリックするとクリップボードにコピーされます。
6
【GA4プロパティが未作成の場合】新規作成する

Analytics にアクセスしてもプロパティが存在しない場合は、以下の手順で作成します。

  1. 管理 → プロパティを作成 をクリック
  2. プロパティ名（例: hide の部屋）・タイムゾーン（日本）・通貨（JPY）を入力 → 次へ
  3. 業種「趣味・レジャー」、規模「小規模」を選択 → 次へ
  4. 「ウェブ」を選択
  5. サイトURL: hifukasawa77-lgtm.github.io/main/ → ストリーム名: hide の部屋 → 作成して続行
  6. 作成後、測定IDが表示される（手順5の画面と同じ）
測定IDが取得できたら、Claude Code に「G-XXXXXXX です」とお知らせください。
index.html のプレースホルダーを自動的に置き換えます。
