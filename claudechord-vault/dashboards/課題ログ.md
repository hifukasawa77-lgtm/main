# 🗂️ 課題ログ（Issue Log）

> 課題は `#issue` タグ付きのチェックボックスで管理する。Tasks プラグイン併用を推奨。
> 記法例: `- [ ] 公害値の計算式を見直す #issue [project:: city_builder]`

## 未対応の課題（オープン）

```dataview
TASK
FROM "claudechord-vault"
WHERE !completed AND contains(text, "#issue")
```

## 対応済みの課題（クローズ）

```dataview
TASK
FROM "claudechord-vault"
WHERE completed AND contains(text, "#issue")
```
