# ⚠️ リスク登録簿（Risk Register）

> legal-checker のリスク報告書（`type: リスク報告書`）を集約。RED/YELLOW を優先表示。

## 要対応リスク（RED / YELLOW）

```dataview
TABLE project AS "PJ", risk_level AS "レベル", target_file AS "対象", updated AS "更新"
FROM "claudechord-vault/deliverables"
WHERE type = "リスク報告書" AND risk_level != "GREEN"
SORT risk_level ASC, updated DESC
```

## レベル別 件数

```dataview
TABLE length(rows) AS "件数"
FROM "claudechord-vault/deliverables"
WHERE type = "リスク報告書"
GROUP BY risk_level AS "レベル"
```

## 全リスク報告書

```dataview
TABLE project AS "PJ", risk_level AS "レベル", status AS "状態", updated AS "更新"
FROM "claudechord-vault/deliverables"
WHERE type = "リスク報告書"
SORT updated DESC
```
