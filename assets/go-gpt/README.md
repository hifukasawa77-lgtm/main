# GPT-image2.0 AI Go Assets

Generated asset pack for `go.html`.

## Files

- `board-surface-gpt-image-2.png`: kaya wood Go board surface used by the canvas renderer.
- `tatami-texture-gpt-image-2.png`: tatami texture used behind the board.
- `go-thumb-gpt-image-2.png`: homepage/search card thumbnail.
- `sprites/stone-black.png`: transparent black stone sprite.
- `sprites/stone-white.png`: transparent white stone sprite.
- `sprites/stones-sheet.png`: transparent stone sprite sheet.
- `asset-preview.png`: preview contact sheet.
- `manifest.json`: machine-readable import map.
- `raw/*.png`: original generated sources.

The stone sprites were generated on a chroma-key background and converted locally to alpha PNGs.
