Use case: logo-brand
Asset type: 2x2 browser game kamon crest atlas for a Sengoku strategy game UI
Primary request: Create four Japanese samurai clan crest marks inspired by historical kamon design language for these families: Bessho clan, Ichijo clan, Daihoji clan, Osaki clan.
Style/medium: flat vector-like mon crest artwork, crisp symmetrical linework, high-contrast ivory-white marks on a pure matte black background, suitable for circular in-game icons.
Composition/framing: square atlas divided conceptually into a 2x2 grid with no visible grid lines. Top-left cell: Bessho. Top-right cell: Ichijo. Bottom-left cell: Daihoji. Bottom-right cell: Osaki. Each crest centered in its own quadrant with generous padding and equal scale.
Design direction: each crest should feel distinct and historically plausible: elegant geometric floral/leaf/circle motifs, strong radial symmetry, simple enough to remain legible at small UI size.
Constraints: no text, no labels, no numbers, no watermark, no signatures, no borders, no UI elements, no visible grid lines, no gradients, no shadows. Keep the background uniformly pure black across the whole image.
