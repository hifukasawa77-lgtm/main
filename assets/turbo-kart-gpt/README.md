# Turbo Kart GPT Image Assets

These raster assets were generated with the built-in GPT image workflow for
`kart_racer.html`.

- `asset-atlas-gpt-image-2.png`: karts, item sprites, trackside props, boost/oil effects, and UI parts.
- `track-textures-gpt-image-2.png`: sky, horizon, asphalt, curb, checker, grass, light, and dust texture panels.

The game keeps its original Canvas/GameKit drawings as a fallback if an image
fails to load, but uses these assets first during normal play.
