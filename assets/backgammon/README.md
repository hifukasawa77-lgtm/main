# GPT-image2.0 Backgammon Assets

Generated asset pack for a backgammon UI.

## Files

- `board.png`: full top-down board background.
- `sprites/checkers-sheet.png`: transparent checker sprite sheet.
- `sprites/dice-ivory-sheet.png`: transparent ivory dice sprite sheet.
- `sprites/dice-burgundy-sheet.png`: transparent burgundy dice sprite sheet.
- `sprites/doubling-cube-sheet.png`: transparent doubling cube sprite sheet.
- `sprites/items/*.png`: individually cropped transparent PNG assets.
- `asset-preview.png`: preview contact sheet.
- `manifest.json`: machine-readable import map.
- `raw/*.png`: original generated sources, including chroma-key versions.

## Individual Assets

- Checkers: `checker-ivory.png`, `checker-burgundy.png`
- Ivory dice: `die-ivory-1.png` through `die-ivory-6.png`
- Burgundy dice: `die-burgundy-1.png` through `die-burgundy-6.png`
- Doubling cube: `doubling-cube-2.png`, `doubling-cube-4.png`, `doubling-cube-8.png`, `doubling-cube-16.png`, `doubling-cube-32.png`, `doubling-cube-64.png`

## Notes

The transparent PNGs were generated on a flat chroma-key background and locally converted to alpha. The original generated files are preserved in `raw/`.
