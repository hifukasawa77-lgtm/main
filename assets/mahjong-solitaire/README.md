# Mahjong Solitaire GPT-image-2 Assets

- `gpt-image-2-thumb.png`: homepage portfolio thumbnail.
- `gpt-image-2-board.png`: canvas board background used by `mahjong-solitaire.html`.
- `gpt-image-2-tile-base-source.png`: original GPT-image-2 chroma-key blank tile source.
- `tiles/tile-base.png`: transparent blank tile base derived from the GPT-image-2 source.
- `tiles/*.png`: game-ready tile images used by the Mahjong Solitaire renderer.

The tile body, board surface, and thumbnail were generated with GPT-image-2. Tile face markings are composited locally into PNG assets for gameplay readability and matching accuracy.
