AI Team
Members

hideの開発を支える10名のAIエージェントたち。
企画・設計・実装・検証・調査まで、チームで動いています。

プランナー
Planner
企画・設計

要件ヒアリングから基本設計・詳細設計まで担当。チームの起点となる司令塔。

デザイナー
Designer
グラフィック

外部ツールやフリー素材を活用して、ゲームやUIのビジュアルを制作する。

ジェネレーター
Generator
実装

仕様書をもとにHTML/CSS/JSを実装。大規模タスクは並行分割で対応する。

エバリュエーター
Evaluator
検証・採点

成果物を100点満点で採点。合格基準80点以上を厳格に判定し品質を保証する。

テスター
Tester
テスト

仕様書との照合・静的解析によりテストケースを作成・実行し、FAILをGeneratorにフィードバック。

セキュリティ
Security
セキュリティ

XSS・SQLインジェクション・SRI未設定などの脆弱性を検査し、修正案を提示する。

リサーチャー
Researcher
市場調査

「まだ存在しないアプリ」を発掘する調査専門家。SNS・フォーラム・App Storeを横断して分析。

デバッガー
Debug Agent
バグ診断

コードを静的解析してバグ・パフォーマンス・セキュリティ・コード品質の問題を診断する。

スペックエージェント
Spec Agent
仕様書作成

ゲームアイデアから要件定義・基本設計・詳細設計・コードスケルトンを一括生成する。

アチーブメント
Achievement Agent
実績称号

ゲームメカニクスを読み解き、ユニークでユーモラスな実績称号を20個生成するムードメーカー。
