Members

hideの開発を支える13名のAIエージェントたち。
企画・設計・実装・検証・法務・音楽まで、チームで動いています。

Members

PMO
PMO Agent
プロジェクト管理

プロジェクト全体を横断監視するPMサポート役。進捗・リスク・課題・KPIを一元管理し、先読みで報告する。

プランナー
Planner
企画・設計

要件ヒアリングから基本設計・詳細設計まで担当。チームの起点となる司令塔。

グラフィックデザイナー
Graphic Designer
グラフィック

外部ツールやフリー素材を活用してゲーム・UIのビジュアルアセットを制作し、Code Generatorへ渡す。

ミュージックジェネレーター
Music Generator
音楽・SE

BGM・SE・ジングルを無料素材またはWeb Audio APIで制作。ゲームに音の命を吹き込む音楽担当。

コードジェネレーター
Code Generator
実装

仕様書をもとにHTML/CSS/JSを実装。グラフィックと音楽の納品物も組み込む。大規模タスクは並行分割で対応。

リーガルチェッカー
Legal Checker
法務チェック

著作権・ライセンス・利用規約の法務リスクを検査。RED/YELLOW/GREENの3段階で判定・報告する。

エバリュエーター
Evaluator
検証・採点

成果物を100点満点で採点。合格基準80点以上を厳格に判定し、品質を保証してGitHubへプッシュする。

セキュリティ
Security
セキュリティ

XSS・SQLインジェクション・SRI未設定などの脆弱性を検査し、修正案を提示する。

リサーチャー
Researcher
市場調査

「まだ存在しないアプリ」を発掘する調査専門家。SNS・フォーラム・App Storeを横断して分析。

デバッガー
Debug Agent
バグ診断

コードを静的解析してバグ・パフォーマンス・セキュリティ・コード品質の問題を診断する。

スペックエージェント
Spec Agent
仕様書作成

ゲームアイデアから要件定義・基本設計・詳細設計・コードスケルトンを一括生成する。

アチーブメント
Achievement Agent
実績称号

ゲームメカニクスを読み解き、ユニークでユーモラスな実績称号を20個生成するムードメーカー。

マーケター
Marketer
マーケティング

完成成果物の市場投入を担う。競合調査・ターゲット分析・KPI設定からX投稿・PR文・キャッチコピーの生成まで一貫対応。
