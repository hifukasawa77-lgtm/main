AI Video Studio

1行のプロンプトで、AIがビジュアル・音声・字幕・構成を生成。YouTube / Shorts / TikTok / Reels 対応
Cloudflare Workers AI · APIキー不要 / no API key

① プロンプト Prompt

🔑 高品質モード（OpenAI BYOK）
キーはこのブラウザ内だけで使用し、リポジトリ・サーバーには保存しません。OpenAIへ直接通信し、台本=gpt-4o-mini / 画像=gpt-image-1 / 音声=gpt-4o-mini-tts を使用します（従量課金はご自身のOpenAIアカウントに発生）。
詳細設定 / Advanced
AIモードは Cloudflare Workers AI（gemma / flux / melotts）を使用します。GitHub Pages 上で動作。ローカルファイルから開くとCORSで拒否されることがあります。

② プロダクト情報（任意） Product

商品名を入れると、その商品を訴求する商品紹介動画になります（商品画像はタイトル・エンディングに実写そのまま表示、AI生成画像と差し替え）。空欄ならこれまで通りの汎用動画を生成します。

③ 台本・絵コンテ（編集可） Storyboard

プロンプトを入れて「AIで動画を生成」を押してください。

④ プレビュー & 書き出し Preview & Export

REC
プロンプトから、AIがビジュアル・音声・字幕・シーン構成を生成します。
⬇ ダウンロード
AIモードでは 映像（AI画像）・字幕・ナレーション音声・BGM がすべて書き出し動画に含まれます。書き出しはリアルタイム録画のため、動画尺と同じ時間がかかります。AI生成に失敗した部分は自動でプロシージャル映像にフォールバックします。

⑤ YouTube 用メタデータ AI auto-generated

タイトル / Title
説明 / Description
タグ / Tags
YouTube への投稿は手動です（キー不要のため）。上のメタデータをコピーして、書き出した .webm と一緒にアップロードしてください。
