🧠 AI/機械学習 用語集

機械学習の基礎概念から、LLM・生成AIに関する用語、AIエージェントを支えるハーネス関連の用語、さらにはハルシネーションやシンギュラリティといったリスク・将来像に関する言葉まで、AI分野でよく使われる用語をまとめて解説する。

📌 機械学習の基礎

用語解説
教師あり学習
(Supervised Learning)		入力データと正解ラベルのペアを使ってモデルを学習させる手法。分類・回帰タスクで使われる。
教師なし学習
(Unsupervised Learning)		正解ラベルなしのデータから構造やパターンを発見する手法。クラスタリングや次元削減など。
強化学習
(Reinforcement Learning)		エージェントが環境とやり取りしながら得る「報酬」を最大化するように行動を学習する手法。
ニューラルネットワーク
(Neural Network)		脳の神経細胞を模した数理モデル。層状に並んだノードが重み付きの結合で接続され、入力から出力を計算する。
ディープラーニング
(Deep Learning)		多数の層を持つニューラルネットワークを用いた機械学習。画像認識・自然言語処理などで大きな性能向上をもたらした。
過学習
(Overfitting)		学習データに過度に適合してしまい、未知のデータに対する性能（汎化性能）が低下する現象。
勾配降下法
(Gradient Descent)		損失関数（誤差）を最小化するために、パラメータを勾配の方向に少しずつ更新していく最適化アルゴリズム。
バックプロパゲーション
(Backpropagation)		出力の誤差をネットワークの逆方向に伝播させ、各層の重みをどれだけ調整すべきかを計算するアルゴリズム（誤差逆伝播法）。

📌 LLM・生成AI用語

用語解説
トークン
(Token)		LLMが処理するテキストの最小単位。単語そのものではなく、単語の一部や文字のまとまりに分割されることが多い。
パラメータ数
(Parameters)		モデル内部の学習可能な重みの総数。モデルの規模を表す指標で、数十億〜数兆に及ぶ。
事前学習
(Pre-training)		大量のテキストデータから言語の統計的パターンを学習するモデル構築の初期段階。
ファインチューニング
(Fine-tuning)		事前学習済みモデルを、特定のタスクやドメインに適応させるための追加学習。
RLHF
(人間のフィードバックからの強化学習)		人間による評価・フィードバックを報酬として用いる強化学習で、モデルの応答を人間の意図や好みに合わせる手法。
プロンプトエンジニアリング
(Prompt Engineering)		LLMから望ましい出力を得るために、入力（プロンプト）の構成や表現を工夫する技術。
RAG
(Retrieval-Augmented Generation)		外部の知識ベースや文書から関連情報を検索し、それをコンテキストに含めて回答を生成する手法。最新情報や専門知識の補完に有効。
Few-shot / Zero-shotプロンプト内にいくつかの例を示してタスクを実行させる手法（Few-shot）と、例を示さずに実行させる手法（Zero-shot）。
Transformer / Attention現代の主要なLLMの基盤となるニューラルネットワーク構造。Attention機構により、入力中の重要な部分に重みづけして処理する。
コンテキストウィンドウ
(Context Window)		モデルが一度の処理で扱える入力＋出力の合計トークン数の上限。
量子化
(Quantization)		モデルの重みをより少ないビット数（低精度）の数値で表現し、メモリ使用量や計算コストを削減する技術。
マルチモーダル
(Multimodal)		テキストだけでなく、画像・音声・動画など複数の種類のデータを入出力として扱えるモデルの性質。

📌 エージェント・ハーネス関連用語

用語解説
エージェント
(Agent)		与えられた目標に対し、自律的に判断・行動（ツール実行など）を繰り返して達成を目指すAIシステム。
ハーネス
(Harness)		エージェントを動作させるための実行基盤・周辺システム全体。CLIや権限管理、ツール実装などを含む「モデルを動かす土台」。
エージェントループ
(Agentic Loop)		「思考 → 行動（ツール呼び出し） → 観察」を繰り返しながらタスクを進める処理の流れ。
ツールユース
(Tool Use)		LLMが外部の関数やAPIを呼び出すことで、情報取得やファイル操作・コマンド実行などを行う仕組み。
MCP
(Model Context Protocol)		LLMアプリケーションと外部ツール・データソースを接続するための標準プロトコル。
サブエージェント / マルチエージェント複数のAIエージェントが役割分担・並列実行・連携しながらタスクを遂行する構成。
オーケストレーション
(Orchestration)		複数のエージェントやツール呼び出しについて、実行順序や役割分担を制御・調整すること。

📌 リスク・限界・将来像

用語解説
ハルシネーション
(Hallucination)		LLMが事実に基づかない、または誤った情報を、自信を持って（あたかも正しいかのように）生成してしまう現象。
アライメント
(Alignment)		AIの目標・行動を人間の意図や価値観に一致させるための研究・技術分野。
プロンプトインジェクション
(Prompt Injection)		悪意のある指示を入力やWebページ等のコンテンツに埋め込み、AIの本来の指示を上書き・操作しようとする攻撃手法。
シンギュラリティ
(Singularity, 技術的特異点)		AIが人間の知能を超え、技術発展が人間の理解や制御を超えて急激に加速するとされる仮説的な時点を指す概念。
AGI
(汎用人工知能)		特定のタスクに限らず、人間と同等以上の汎用的な知的能力を持つとされるAIの概念（Artificial General Intelligence）。
スケーリング則
(Scaling Laws)		モデルサイズ・学習データ量・計算量を増やすほど、性能が予測可能な形で向上するという経験的な法則。
レッドチーミング
(Red Teaming)		AIシステムの脆弱性や有害な出力を発見するために、意図的に敵対的・攻撃的な入力でテストを行う取り組み。
